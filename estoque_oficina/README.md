🛠 Gerenciador de Estoque - Oficina

📌 Descrição

Sistema desenvolvido em Python para gerenciamento de estoque de uma oficina mecânica.

🚀 Funcionalidades

- Cadastro de produtos
- Consulta de produtos
- Visualização completa do estoque
- Exportação de relatório em PDF

🛠 Tecnologias utilizadas

- Python
- ReportLab

▶️ Como executar o projeto

1. Instale as dependências:

pip install reportlab

2. Execute o programa:

python main.py

📄 Observação

O arquivo PDF é gerado automaticamente ao utilizar a opção "exportar pdf" no sistema.
